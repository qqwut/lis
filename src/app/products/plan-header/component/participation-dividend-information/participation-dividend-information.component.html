Participation Indicator

(PLAN_PAR_CD)

This field is used to determine whether dividends should be paid if participating premium rates are used. Participation codes 'N' and 'P' are used on the coverage level, and this field will determine the appropriate coverage default.
    Allowed Value Description
  • 0 No dividend paid
  • 1 Dividend paid on anniversary date if the policy is paid to the anniversary date; default value
  • 2 Reserved for future use
  • 3 Reserved for future use

(DOD_INT_RT_ID)

6 character key used to access interest rates on the IR table for dividends on deposit. If this is not a plan code/rate scale, it must be present in the 'EDIT' table. Default value is 'DIV 1'. Edited against the ‘EDIT’ table with a type of ‘IRTS’.
