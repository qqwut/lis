ETI Face Amount

(ETI_FACE_CALC_CD)

This field controls the method of calculation used to determine the face amount of extended term insurance (ETI).
    Allowed Value Description
  • 1 Face amount minus loan; default value
  • 2 to 4 Reserved for future enhancement
  • 5 Sum insured amount minus loan
ETI Calculation

(ETI_TBAC_ID)

The plan code this plan converts to for extended term insurance. First two characters give MxDx table to use when calculating the cash value of the ETI coverage. If ** is used as the leading characters, it means use UVAL (for original plan code) instead of MxDx. Default value is 'NO'. GUI selection box created from the ‘EDIT’ table with a type of ‘MXDX’. RPU Calculation

(RPU_TBAC_ID)

The field indicates the plan code to which this plan converts to for RPU (reduced paid-up). The first two positions are used in conjunction with the age to create the key to MxDx which determines the cash value of RPU coverages. If the first two positions in the field are '**', then NSP (net single premium) in UVAL will be used under the UVAL pointer of the original plan. Any other values in the first two positions of the field will point to MxDx. GUI selection box created from the ‘EDIT’ table with a type of ‘MXDX’. ETI/RPU-Exceed NSP Calc
