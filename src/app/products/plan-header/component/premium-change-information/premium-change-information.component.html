Premium Change Calc.

(PREM_CHNG_DT_CD)

This field is used to describe the type of premium change if one is to occur.
    Allowed Value Description
  • 0 No premium change; default value
  • A Premium changes to zero in a given number of years (policy becomes paid-up)
  • B Premium changes to zero at a given attained age (policy becomes paid-up)
  • D Modification plan - premium changes at a given number of years, no auto change
  • E Premium changes every x years, starting/ending at a given attained age
Age/Year

(PREM_CHNG_AGE_DUR)

This field defines the number of years or age at which the premium changes as defined by the premium change type field. If the premium change code is A or D, then enter the number of years at which the premium changes. If the premium change code is B, then enter the age at which the premium changes If the premium change code is C or E, then enter the number of years between changes A value of zero is used if there is not a premium change; default value.
